Git help

Main Porcelain Commands

  1. add: Add file contents to the index
  2. am: Apply a series of patches from a mailbox
  3. archive: Create an archive of files from a named tree
  4. bisect: Use binary search to find the commit that introduced a bug
  5. branch: List, create, or delete branches
  6. bundle: Move objects and refs by archive
  7. checkout: Switch branches or restore working tree files
  8. cherry-pick: Apply the changes introduced by some existing commits
  9. citool: Graphical alternative to git-commit
  10. clean: Remove untracked files from the working tree
  11. clone: Clone a repository into a new directory
  12. commit: Record changes to the repository
  13. describe: Give an object a human readable name based on an available ref
  14. diff: Show changes between commits, commit and working tree, etc
  15. fetch: Download objects and refs from another repository
  16. format-patch: Prepare patches for e-mail submission
  17. gc: Cleanup unnecessary files and optimize the local repository
  18. gitk: The Git repository browser
  19. grep: Print lines matching a pattern
  20. gui: A portable graphical interface to Git
  21. init: Create an empty Git repository or reinitialize an existing one
  22. log: Show commit logs
  23. maintenance: Run tasks to optimize Git repository data
  24. merge: Join two or more development histories together
  25. mv: Move or rename a file, a directory, or a symlink
  26. notes: Add or inspect object notes
  27. pull: Fetch from and integrate with another repository or a local branch
  28. push: Update remote refs along with associated objects
  29. range-diff: Compare two commit ranges (e.g. two versions of a branch)
  30. rebase: Reapply commits on top of another base tip
  31. reset: Reset current HEAD to the specified state
  32. restore: Restore working tree files
  33. revert: Revert some existing commits
  34. rm: Remove files from the working tree and from the index
  35. scalar: A tool for managing large Git repositories
  36. shortlog: Summarize 'git log' output
  37. show: Show various types of objects
  38. sparse-checkout: Reduce your working tree to a subset of tracked files
  39. stash: Stash the changes in a dirty working directory away
  40. status: Show the working tree status
  41. submodule: Initialize, update or inspect submodules
  42. switch: Switch branches
  43. tag: Create, list, delete or verify a tag object signed with GPG
  44. worktree: Manage multiple working trees

Ancillary Commands / Manipulators

  1. config Get and set repository or global options
  2. fast-export Git data exporter
  3. fast-import Backend for fast Git data importers
  4. filter-branch Rewrite branches
  5. mergetool Run merge conflict resolution tools to resolve merge conflicts
  6. pack-refs Pack heads and tags for efficient repository access
  7. prune Prune all unreachable objects from the object database
  8. reflog Manage reflog information
  9. remote Manage set of tracked repositories
  10. repack Pack unpacked objects in a repository
  11. replace Create, list, delete refs to replace objects

Ancillary Commands / Interrogators

  1. annotate Annotate file lines with commit information
  2. blame Show what revision and author last modified each line of a file
  3. bugreport Collect information for user to file a bug report
  4. count-objects Count unpacked number of objects and their disk consumption
  5. diagnose Generate a zip archive of diagnostic information
  6. difftool Show changes using common diff tools
  7. fsck Verifies the connectivity and validity of the objects in the database
  8. gitweb Git web interface (web frontend to Git repositories)
  9. help Display help information about Git
  10. instaweb Instantly browse your working repository in gitweb
  11. merge-tree Perform merge without touching index or working tree
  12. rerere Reuse recorded resolution of conflicted merges
  13. show-branch Show branches and their commits
  14. verify-commit Check the GPG signature of commits
  15. verify-tag Check the GPG signature of tags
  16. version Display version information about Git
  17. whatchanged Show logs with differences each commit introduces
